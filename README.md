# 低代码平台权限设计项目

## 📋 项目概述

本项目为低代码平台设计完整的权限体系，支持灵活的功能权限和数据权限。

### 设计背景
1. 低代码平台支持配置和发布多个应用，配置和发布在一个平台底座中，但通过不同的URL访问
2. 发布后的应用，由该应用的运营管理员负责人员角色权限的设置
3. 低代码平台有页面设计器、实体设计器、流程设计器等功能
4. 平台使用 K8s + Java17 + Spring Boot 3 技术栈

### 设计要求
1. 应用配置阶段支持设置应用的权限配置项
2. 发布后支持灵活配置权限
3. 定义权限配置的 DSL
4. 设计权限生效方式

## 🎯 核心特性

| 特性 | 说明 |
|------|------|
| **权限模型** | RBAC + ABAC 混合模型 |
| **功能权限** | 页面 / 按钮 / 字段 / API 四级权限 |
| **数据权限** | 组织 / 部门 / Owner / 自定义维度 |
| **配置分离** | 配置期 DSL + 运行时 DSL |
| **高性能** | 多级缓存，毫秒级响应 |
| **安全可靠** | 数据脱敏 / API限流 / 审计日志 |
